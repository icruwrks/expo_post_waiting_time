# 概要

- 大阪万博パビリオン待ち時間投稿サイト
- カテゴリ、パビリオン、待ち時間をそれぞれプルダウンリストで選択でき、投稿するボタンがあるHTML
- 投稿するボタンを押すと、Discordにその内容が送信される

# 待ち時間

- 時間と、5分単位で選択できるプルダウン〇時間〇〇分みたいな（0も可能）
- 一時停止中、自由入場終了、午前の部受付終了、受付終了、予約のみ

# 投稿先

- 設定したディスコードチャンネルに投稿される

# 投稿内容例

- 日本館　午前の部受付終了
- イタリア　5時間以上
- トルクメニスタン 25分
- アゼルバイジャン待ちなし
